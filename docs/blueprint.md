# **App Name**: Global Career Pathways

## Core Features:

- Homepage Hero Section: Display a realistic image of Indian professionals working in diverse overseas environments to build trust and increase scroll rate.
- Countries Section: Use skyline and flag images in a minimal style for quick country recognition, enhancing user experience without text-heavy descriptions.
- Visual Process Illustrations: Illustrate the job application process in four steps (eligibility, guidance, documentation, offer) using vector graphics for clear, instant understanding.
- Success Stories Section: Showcase success stories with realistic workplace photos, name, country, and job title, using a before/after structure to convert doubt into action.
- Resume Analyzer: Employ an AI-driven tool to offer constructive feedback on resume uploads, enhancing user confidence in their application documents.

## Style Guidelines:

- Primary color: Deep blue (#1A5276) to convey trust and professionalism, mirroring the reliability desired by job seekers.
- Background color: Very light blue (#EBF5FB), a desaturated version of the primary color, providing a clean, unobtrusive backdrop.
- Accent color: Soft green (#A9D08E), analogous to blue, used for call-to-action buttons and highlighting key information to guide the user.
- Body and headline font: 'Inter', a sans-serif font chosen for its modern, neutral, and readable qualities, suitable for both headlines and body text, promoting clarity and ease of reading.
- Employ industry-relevant icons for job categories and eligibility indicators to make the platform intuitive and engaging, helping users visualize opportunities.
- Maintain a clean and clear image + text hierarchy for each job and country for mobile-first image scaling
